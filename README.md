# WebGL based simple image filter With EasyLogic


# Install 

## npm 

```npm
npm install @easylogic/gl-filter
```

# How to use 
 
```
import GLFilter from '@easylogic/gl-filter' 
```


# Developments 

## local dev 

```
git clone https://github.com/easylogic/gl-filter
cd image-filter
npm install 
npm run dev 
open localhost:3001
```

## build 

```
npm run build 
```
